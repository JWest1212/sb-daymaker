  • 3 panels, in narrative order: what it is → save (want) → remember (been). Panel 3 plants "been" before the user can do it, so the native "Did you make it?" prompt lands later instead of confusing.
  • The heart-pop and glowing sun are the existing signature micro-interactions, reused, not new decoration.
  • Skip is always visible. Panel 3's primary action becomes Show me today → and drops them straight into the feed.
  • Replayable: a quiet "Replay the intro" link sits in the Explore footer (shown in the phones below), so it's never a dead end.
  • Reduced motion: toggle it on and the sheet, sun-pulse, heart-pop and panel fades all go static; swipe/dots still work.
