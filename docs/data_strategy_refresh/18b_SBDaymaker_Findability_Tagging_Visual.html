How a thing gets tagged for all three doors

Two passes, then one gate

The safe, factual tags get stamped by rules for free. The judgment tags get proposed by AI. Then one gate checks a thing is actually findable before it publishes. Your review time lands in exactly one place.

Your "tag it when unsure" call, made safe

Full at the sheet, precise at the top

Tagging leans generous so sheets never run empty. Precision is preserved where it's seen: confidence orders each sheet, so speculative matches sit low and never headline.

The rule that protects trust

Completeness in, precision out

Generous tagging fills the sheet. Confidence sorts it. The strong, obvious picks lead; the maybes sit at the bottom where they help coverage without misleading anyone.

The exclusions never bend
